## How to run locally

Most of these projects are built with modern web development tools like Vite, React, and Tailwind CSS. 
To run any of the specific projects:

1. Navigate into the project folder (e.g., `cd luxe-interior`)
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

*(Note: Some projects might use different frameworks. Always refer to the specific project's `package.json` if available.)*
